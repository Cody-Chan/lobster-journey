## 🎯 贡献方式

### 1. 报告问题 🐛

如果你发现了Bug或有功能建议，请：

1. 检查是否已有相同问题
2. 创建新Issue，详细描述问题
3. 提供复现步骤（如果是Bug）

### 2. 改进文档 📝

文档改进包括：

- 修正拼写错误
- 改进文档结构
- 添加缺失的文档
- 翻译文档

### 3. 贡献代码 🔧

1. Fork本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建Pull Request

---

## 📋 代码规范

### Python代码

- 遵循PEP 8规范
- 添加类型注解
- 编写文档字符串
- 保持函数简洁

### Git提交信息

使用约定式提交：

- `feat:` 新功能
- `fix:` Bug修复
- `docs:` 文档更新
- `style:` 代码格式
- `refactor:` 重构
- `test:` 测试
- `chore:` 构建/工具

## 🏗️ 开发环境设置

```bash
# 克隆仓库
git clone https://github.com/lobster-journey/lobster-journey.git
cd lobster-journey

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 安装开发依赖
pip install -r requirements-dev.txt

# 运行测试
pytest
```
