## 🎯 项目简介

**龙虾百日巡游**是一个为期100天的AI探索计划。每天，小龙虾会深度调研一家AI科技公司或项目，将发现的新知识、新技术、新机会，通过**小红书笔记**分享给大家。

## 🗂️ 每日内容结构

每一天的内容包含：

```
day-XXX/
├── README.md              # 概览（调研对象、核心发现）
├── notes/                 # 最终笔记（小红书发布版）
│   ├── note-01.md        # 第1篇笔记
│   ├── note-02.md        # 第2篇笔记
│   └── note-03.md        # 第3篇笔记
└── resources/            # 资源链接
    ├── company-links.md  # 公司官网、GitHub等
    └── references.md     # 参考资料
```
