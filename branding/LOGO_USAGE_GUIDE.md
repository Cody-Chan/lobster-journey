## 📐 技术要求

### 小红书头像

**推荐图片**：`logo-option-01.jpg`

**技术规格**：
- 格式：JPG 或 PNG
- 尺寸：400x400px（正方形）
- 大小：< 5MB
- 小红书会自动裁剪为圆形

**操作步骤**：
1. 打开小红书APP
2. 点击"我" → 编辑资料
3. 点击头像位置
4. 选择 `logo-option-01.jpg`
5. 调整裁剪位置（确保主体居中）
6. 保存

---

### 小红书背景图

**推荐图片**：`logo-option-02.jpg`

**技术规格**：
- 格式：JPG
- 尺寸：1920x1080px（16:9横版）
- 大小：< 5MB

**操作步骤**：
1. 在编辑资料页面
2. 点击背景图位置
3. 选择 `logo-option-02.jpg`
4. 调整位置
5. 保存

---

### GitHub头像

**推荐图片**：`logo-option-01.jpg`

**操作步骤**：
1. 登录 GitHub
2. 点击右上角头像 → Settings
3. 点击 Profile
4. 上传 `logo-option-01.jpg`
5. 调整裁剪位置
6. 保存
